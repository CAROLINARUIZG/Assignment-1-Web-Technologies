# Historias de Usuario - EventHub

1. **Como Usuario**, quiero **ver una lista de eventos**, para **enterarme de las actividades del campus**.
2. **Como Organizador**, quiero **crear un evento con aforo limitado**, para **controlar la asistencia**.
3. **Como Usuario**, quiero **unirme a una lista de espera**, para **tener oportunidad de asistir si alguien cancela**.
4. **Como Administrador**, quiero **moderar contenido**, para **mantener la plataforma segura**.
