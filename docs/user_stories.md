# Historias de Usuario - EventHub

### 1. Creación y Gestión de Eventos
* **Como Organizador**, quiero **crear un evento con título, descripción enriquecida, categoría, lugar y fechas**, para **proporcionar toda la información necesaria a los interesados.**
* **Como Organizador**, quiero **establecer un aforo máximo**, para **asegurar que la asistencia no supere la capacidad del lugar.**
* **Como Organizador**, quiero **editar o cancelar mis propios eventos**, para **mantener la información actualizada o suspender la actividad si es necesario.**

### 2. Descubrimiento y Búsqueda de Eventos
* **Como Usuario**, quiero **ver la lista de todos los eventos publicados**, para **enterarme de las próximas actividades en el campus.**
* **Como Usuario**, quiero **filtrar eventos por categoría**, para **encontrar rápidamente actividades de mi interés.**
* **Como Usuario**, quiero **ver los detalles de un evento (lugar, plazas, reseñas)**, para **decidir si deseo asistir.**

### 3. Registro y Lista de Espera
* **Como Usuario**, quiero **inscribirme en un evento**, para **reservar mi lugar.**
* **Como Usuario**, quiero **ser incluido en una lista de espera si el evento está lleno**, para **obtener un cupo automáticamente si alguien cancela su asistencia.**
* **Como Usuario**, quiero **cancelar mi inscripción**, para **liberar mi cupo si ya no puedo asistir.**

### 4. Revisión y Calificación
* **Como Asistente**, quiero **dejar una calificación de 1 a 5 y un comentario sobre un evento finalizado**, para **ayudar a otros a evaluar futuras actividades similares.**

### 5. Acciones Administrativas (Moderación)
* **Como Administrador**, quiero **editar o eliminar cualquier evento, lugar o categoría**, para **mantener la plataforma organizada y moderada.**
