# User Stories — EventHub

### 1. Event Creation and Management
* **As an Organizer**, I want to **create an event with a title, rich description, category, location, and dates**, so that **I can provide all the necessary information to interested attendees.**
* **As an Organizer**, I want to **set a maximum capacity for my event**, so that **attendance does not exceed the venue's limit.**
* **As an Organizer**, I want to **edit or cancel my own events**, so that **I can keep the information up to date or suspend the activity if needed.**

### 2. Event Discovery and Browsing
* **As a User**, I want to **view a list of all published events**, so that **I can learn about upcoming activities on campus.**
* **As a User**, I want to **filter events by category**, so that **I can quickly find activities that interest me.**
* **As a User**, I want to **view the details of an event (location, available spots, reviews)**, so that **I can decide whether I want to attend.**
* **As a User**, I want to **search events by keyword**, so that **I can find specific events without browsing the entire list.**

### 3. Registration and Waiting List
* **As a User**, I want to **register for an event**, so that **I can reserve my spot.**
* **As a User**, I want to **be placed on a waiting list if the event is full**, so that **I can automatically get a spot if someone cancels their registration.**
* **As a User**, I want to **cancel my registration**, so that **I can free up my spot if I can no longer attend.**
* **As a User**, I want to **receive a confirmation of my registration status (confirmed or waiting list)**, so that **I know whether I have a guaranteed spot.**

### 4. Review and Rating of Past Events
* **As an Attendee**, I want to **leave a rating from 1 to 5 and a comment on a finished event**, so that **I can help others evaluate similar future activities.**
* **As a User**, I want to **read reviews and ratings left by other attendees**, so that **I can make informed decisions about attending similar events.**

### 5. Administrative Actions (Moderation)
* **As an Administrator**, I want to **edit or delete any event, venue, or category**, so that **I can keep the platform organized and moderated.**
* **As an Administrator**, I want to **manage user accounts (activate, deactivate, or change roles)**, so that **I can ensure proper access control on the platform.**
* **As an Administrator**, I want to **remove inappropriate reviews**, so that **the platform maintains a respectful and trustworthy environment.**
